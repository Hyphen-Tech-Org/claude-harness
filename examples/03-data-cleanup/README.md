# Example 03 — Data Cleanup with Claude Code

> 想定時間: 10 分
> ゴール: 表記揺れの混じった CSV を、Claude にクレンジングしてもらう

## サンプルデータ

[`sample.csv`](./sample.csv) を見てください。
わざと以下のような揺れを混ぜています:

- 日付フォーマット (`2026/05/13`, `2026-5-13`, `2026年5月13日`)
- 全角 / 半角の混在 (`１２３`, `123`)
- 会社名の表記揺れ (`株式会社A`, `(株)A`, `A株式会社`)
- 半角・全角スペースの混在

## 手順

### 1. 起動

```bash
cd examples/03-data-cleanup
claude
```

### 2. クレンジング依頼

```
sample.csv を読んで、以下を統一して cleaned.csv に書き出して:
- 日付: YYYY-MM-DD
- 数値: 半角
- 会社名: 「株式会社A」のような株式会社が前に来る形に統一
- スペース: 半角に統一

変換ルールを README ではなく、cleaned.csv の隣に CLEANING_RULES.md として書き出しておいて。
```

### 3. 結果の確認

```bash
cat cleaned.csv
cat CLEANING_RULES.md
```

期待通りでない行があれば、追加指示:

```
3 行目だけ会社名の整形に失敗してるよ。「(株)A」を「株式会社A」に直して。
```

### 4. 自分の業務 CSV で試す

`sample.csv` をあなたの実データ（個人情報を除いたもの）で置き換えて、同じことをやってみる。

## なぜ Claude Code でやるか？

- ✅ **規則を自然言語で表現**できる（Excel の関数を覚える必要なし）
- ✅ **例外行**にも柔軟に対応（「3 行目だけ」がそのまま伝わる）
- ✅ 変換ロジックをドキュメント化（CLEANING_RULES.md）できる
- ✅ 再現可能（Skill 化すれば毎週の定型処理に）
