# Contributing

Claude Code Starter Kit へのコントリビューション、ありがとうございます！

## どんな貢献を歓迎しているか

- ✅ ドキュメントの誤字脱字 / 古い情報の修正
- ✅ よくあるエラーと対処（`docs/14-troubleshooting.md`）の追加
- ✅ 新しいハンズオン例（`examples/`）
- ✅ Skills / Subagents / Hooks のサンプル
- ✅ 他言語への翻訳（特に英語）

## 進め方

1. Issue を立てて「何をしたいか」を簡潔に書く（任意だが推奨）
2. Fork して branch を切る — `git checkout -b docs/fix-typo`
3. 変更を commit する — Conventional Commits 推奨 (`feat:` / `fix:` / `docs:`)
4. PR を出す

## コンテンツの方針

- **1 次情報のみ引用する**。Anthropic 公式ドキュメント、公式 GitHub リポ、署名された blog post を優先。
- **古い情報は明示する**。「2026-05 時点」など日付を必ず書く。
- **手を動かして確認したことだけ書く**。動かしていない手順は載せない。

## レビュー基準

- ドキュメントは「コード書いた事ない人」が読んで理解できるか
- リンク切れ / 404 がないか（外部 URL は変わるので注意）
- 引用元が信頼できるか（個人 blog だけ、ではなく公式ソースが裏付けになっているか）

## ライセンス

PR を出した時点で、あなたの貢献は **MIT License** で公開されることに同意したとみなされます。
